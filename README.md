# digitalkovachnica

## NeonHost (Web Hosting Landing Page)

Това е статична landing страница (HTML + inline CSS + JS), с синьо-лилава “neon/space” тематика.

### Как да я отвориш
1. Отвори `index.html` директно с браузър (double click).
2. Или в браузър: File → Open → избери `index.html`.

### Как работят формата и CTA
- Бутоните “Избери план” и “Вземи оферта” правят smooth scroll.
- Формата “Вземи оферта” генерира `mailto:` имейл до `hello@neonhost.bg` с попълнен текст.

### Функции на страницата
- Hero секция
- Plans (Starter / Business / Pro)
- Features
- How it works
- FAQ (details/accordion)
